Web des données et documentaire

Le Web des données est une initiative visant à favoriser la publication de données structurées sur le Web, en évitant de les représenter sous forme d’un gros bloc décorrélé et plutôt en les reliant entre elles pour constituer un réseau global d'informations.

On parle de Web de données par opposition au Web "des origines" : un Web dit documentaire, autrement dit un ensemble de documents interconnectés et accessibles grâce à Internet, à destination des utilisateurs humains qui allaient naviguer d’un document à l’autre pour en traiter eux-mêmes le contenu informatif. Le Web des données 'repose' donc sur le Web documentaire classique.
