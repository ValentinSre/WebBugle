Représentation concrète des données

Comment représenter les données sur ce "Web des données" ?







Notion de triplet

Explication

Comme nous l'avons vu précédemment, les données peuvent être représentées sous forme de triplet.
Le format RDF (Resource Description Framework) est le format utilisé pour décrire les données vues précédemment sur DBPedia. Ce format explicite cette notion de triplet.

Représentation

Les 3 éléments sont les suivants :

  • le sujet = ressource
  • le prédicat = nom de la propriété de la ressource
  • l'objet = valeur = ressource liée ou caractéristique exprimée

Néanmoins, tout ceci pose la question de l'interprétation : il semble difficile de comprendre le lien entre tous ces triplets.
C'est ici qu'intervient la représentation graphique des données !
