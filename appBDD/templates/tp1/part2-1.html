Qu'est-ce qu'un navigateur ?



Réponse

Un navigateur est un logiciel qui permet de consulter des pages Web.
Chrome, Firefox, Safari, Edge en sont des exemples.



Consultation d'un document

Manipulation

Ouvrez un navigateur Web et entrez « www.glenat.com/manga » dans la barre d’adresse.



Réponse

L’adresse est appelée une URL (Uniform Resource Locator). Elle se compose d’un nom du serveur Web qui contient toutes les informations recherchées (glenat.com) et d’une page Web à afficher (/manga).
Grâce au nom du serveur, le navigateur récupère l’adresse IP du serveur et lui demande la page recherchée.



Code source

Le navigateur ne sert pas simplement à afficher une page recherchée : en effet, il interprète le “code source” pour afficher de manière interprétable les véritables données stockées sur les serveurs d’Internet.

Manipulation


Placez votre souris sur la page web et faites un clic droit. Sélectionnez l’option « Code source de la page ». Observez la page et essayez de repérer des éléments reconnaissables.

Réponse

Vous découvrez ce qu’est le HTML ! Il s’agit d’un langage de programmation pour créer et représenter du contenu Web.

La page est stockée dans cet état dans les serveurs. HTML est très utilisé, souvent conjugué à d’autres langages. Il est évidemment difficilement compréhensible pour des non-initiés mais il est notamment essentiel pour un concept bien connu que nous avons vu…



Href

Question

Repérez une ligne « href ».
Que remarquez-vous ?
À quoi sert ce type de ligne selon vous ?

Réponse

Les « href » sont les hyperliens. Si l’on regarde de plus près, on remarque que ces balises « href » sont suivies d’un texte bleu et souligné de la forme « /... » désignant une autre page web du site.
(Par exemple, à la ligne 188, « /comics » : cet hyperlien renvoie vers la page web du site concernant les comics).
