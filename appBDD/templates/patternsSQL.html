Structure d'une requête SQL

SELECT colonne(s)
FROM Table1
JOIN Table2 ON condition(s)
WHERE condition(s)
GROUP BY colonne(s)
HAVING condition(s)
ORDER BY colonne(s) ASC/DESC

Création de tables

CREATE TABLE nomTable (nomColonne1 type, nomColonne2 type, ...)

Remplissage de tables

INSERT INTO Table VALUES (valeur1, valeur2, ...)

Fonctions d'agrégation utiles

SUM() : calcule la somme d'une colonne ou groupe de lignes
AVG() : calcule la moyenne d'une colonne ou groupe de lignes
MAX() : retourne le résultat maximum
MIN() : retourne le résultat minimum
COUNT() : compte le nombre de lignes dans un résultat
ROUND() : arrondit la valeur

ORDER BY

Permet de trier les lignes (ASC = ascendant ; DESC = descendant)

GROUP BY

Permet de grouper plusieurs résultats et utiliser une fonction sur un groupe de résultat

LIMIT

Permet de spécifier le nombre maximum de résultats que l'on souhaite obtenir

HAVING/WHERE

HAVING permet de filtrer après le GROUP BY alors que WHERE filtre avant
