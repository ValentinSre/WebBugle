Le CSS

Un site Internet se compose de plusieurs langages dont l’utilisation de chacun répond à un but bien précis : c’est leur addition qui forme une page Web.


Le style

Comme nous l’avons vu, la page HTML créée est "brute".
Pour rendre cela joli, il faut lui donner un style : c’est le rôle des fichiers CSS (pour Cascading Style Sheets).

Ces fichiers permettent par exemple de définir la position ou la taille d’un élément, la couleur et la taille des polices, la couleur du fond… Il peut être inséré dans un fichier .css ou directement dans le code HTML grâce à la balise < style >.
