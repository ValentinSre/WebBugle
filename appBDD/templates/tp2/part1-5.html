Architecture d'un site

En conclusion, un site Web est en réalité une superposition de plusieurs langages. Chaque langage a un rôle spécifique, permettant d'assurer une fonction précise.

Pour schématiser, un site Web pourrait être comparé à un restaurant.
Les langages HTML, CSS et JavaScript forment l'interface : dans le cas d'un restaurant, il s'agirait de l'aspect "visible", c'est-à-dire la salle avec ses tables, sa décoration, ses employés, ses plats...
Le langage PHP (ou NodeJS ou Python) permet de mettre en place des algorithmes pour "préparer" la page à laquelle on souhaite accéder : dans notre exemple, il s'agirait ici de la cuisine où l'on prépare les plats.
En somme, il nous manque un élément essentiel au bon fonctionnement du site (ou de notre restaurant) : les données !
