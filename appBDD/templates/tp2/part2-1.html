Base de données

Explication

Une base de données est un collection de données, généralement stockées et accessibles électroniquement.
Les données sont organisées pour faciliter leur manipulation.

Exemple

Des programmes permettent d'ailleurs aux utilisateurs de gérer une base de données, comme Excel ou MySQL.

Le but d'une BDD est de stocker des données, proposer un mécanisme pour interroger, créer, modifier et supprimer des données et fournir une structure organisationnelle pour les données.
