Le HTML

Un site Internet se compose de plusieurs langages dont l’utilisation de chacun répond à un but bien précis : c’est leur addition qui forme une page Web.


La structure

Le HTML (pour HyperText Markup Language) correspond grossièrement à une description de la page Web, c’est-à-dire qu’il correspond à un fichier définissant la structure de la page Web.

"Markup" signifie "balise", un point essentiel de ce langage. Les balises, repérées entre crochets (< > pour ouvrir, < / > pour fermer), définissent les différentes sections de la page avec leur contenu. Par exemple, la balise < head > définit l’en-tête, la balise < body > contient le corps de la page. Globalement, il y a une balise dédiée à tous les types d’éléments que l’on peut exploiter au sein d’une page web (titre, paragraphe, formulaire, image…).
