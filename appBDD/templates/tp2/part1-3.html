Le JavaScript

Un site Internet se compose de plusieurs langages dont l’utilisation de chacun répond à un but bien précis : c’est leur addition qui forme une page Web.


Le JavaScript est un langage permettant de décrire les interactions entre l’utilisateur et la page Web, mais aussi de modifier certains éléments indépendamment des autres sans avoir à recharger la page.




Dynamise ta page HTML ! 

<button onclick=ouvrirFenetre()>
  Appuie pour ouvrir !
</button>
<script>
function ouvrirFenetre(){
  window.alert("Tu as ouvert cette fenêtre !");
}
</script>

Manipulation

Le code ci-joint est écrit en JavaScript ! Il définit que, si l'on appuie sur le bouton, la fonction ouvrirFenetre() se joue et ouvre une fenêtre. Teste-le !
